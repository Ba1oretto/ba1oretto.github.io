### JDBCUtils
```java
import java.sql.Connection;
import java.sql.DriverManager;

public class JDBCUtils {
    /**
     * 连结MySQL服务器捏
     */
    public static Connection JDBCConnector() throws Exception {
        Class.forName("com.mysql.cj.jdbc.Driver");
        return DriverManager.getConnection("jdbc:mysql://127.0.0.1:3306/test", "root", "baioretto");
    }
}
```

### 测试 表格标签
```html
<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title>测试 表格标签</title>
	</head>
	<body>
		<!-- 1.准备表格
			 结构: table里包含tr行, tr行里包含td列
			 属性: border设置边框, width表格的宽度cellspacing单元格的距离, bgcolor背景色
			 colspan是列合并: 向右合并, 把多个列合并成一个大列, 值是指合并几个
			 rowspan是行合并: 向下合并, 把多个行合并成一个大行, 值是指合并几个
			 -->
		<table
			border
			width="500xp"
			cellspacing="0xp"
			bgcolor="#FFC0CB">
			<tr>
				<td colspan="2">11</td>
				<td>13</td>
			</tr>
			
			<tr>
				<td>21</td>
				<td>22</td>
				<td rowspan="2">23</td>
			</tr>
			
			<tr>
				<td>31</td>
				<td>32</td>
			</tr>
		</table>
	</body>
</html>
```
